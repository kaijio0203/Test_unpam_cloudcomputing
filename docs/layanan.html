Model Layanan Cloud

Beranda Layanan Cloud Profil Penulis

IaaS (Infrastructure as a Service)

Menyediakan infrastruktur komputasi dasar seperti virtual machine, storage, dan jaringan. Pengguna memiliki kontrol penuh atas sistem operasi dan aplikasi.

Contoh: AWS EC2, Google Compute Engine, Microsoft Azure.

PaaS (Platform as a Service)

Menyediakan platform dan lingkungan untuk developer membuat aplikasi. Tidak perlu mengelola OS atau infrastruktur dasar, cukup fokus pada coding.

Contoh: Heroku, Google App Engine.

SaaS (Software as a Service)

Layanan berupa aplikasi yang sudah jadi dan siap pakai melalui internet. Pengguna tidak perlu menginstal atau memelihara perangkat lunak.

Contoh: Google Workspace, Microsoft 365, Dropbox.
