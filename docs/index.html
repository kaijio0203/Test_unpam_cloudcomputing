Mengenal Cloud Computing

Teknologi yang Mengubah Cara Dunia Mengelola Data

Apa itu Cloud Computing?

Secara sederhana, Cloud Computing adalah pengiriman berbagai layanan melalui internet. Sumber daya ini mencakup alat dan aplikasi seperti penyimpanan data, server, database, jaringan, dan perangkat lunak.

Kenapa Cloud Computing Menarik?

On-Demand Self-Service

Pengguna dapat memesan dan mengelola layanan secara otomatis tanpa interaksi manusia dengan penyedia layanan.

Scalability

Kapasitas bisa ditambah atau dikurangi dalam hitungan detik sesuai kebutuhan trafik aplikasi kita.

Pay-as-you-go

Kita hanya membayar apa yang kita gunakan. Tidak perlu investasi besar di awal untuk hardware fisik.

Global Reach

Aplikasi bisa di-deploy ke seluruh dunia hanya dengan beberapa klik saja.

Tipe Layanan Cloud

  • IaaS (Infrastructure as a Service) - Contoh: AWS EC2, Google Compute Engine.
  • PaaS (Platform as a Service) - Contoh: Heroku, Google App Engine.
  • SaaS (Software as a Service) - Contoh: Google Drive, Microsoft 365.
