Selamat Datang

Cloud Computing atau komputasi awan adalah metode penyampaian berbagai layanan melalui internet. Mulai dari penyimpanan data, server, database, hingga perangkat lunak, semuanya bisa diakses tanpa harus mengelola infrastruktur fisiknya secara langsung.

Karakteristik Utama

  • On-Demand Self-Service: Kapasitas IT disediakan secara otomatis.
  • Broad Network Access: Bisa diakses dari mana saja melalui internet.
  • Resource Pooling: Sumber daya dikumpulkan untuk melayani banyak konsumen.
