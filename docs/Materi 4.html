Grafika Komputer: Algoritma Lingkaran dan Elips

1. Algoritma Pembentukan Lingkaran

Lingkaran adalah kumpulan titik-titik yang memiliki jarak yang sama dari titik pusatnya. Penggambaran lingkaran sering dimulai dengan menggambar seperempat lingkaran dan memanfaatkan simetri.

Persamaan lingkaran dengan pusat (x_p, y_p) dan jari-jari r adalah: 

(x-x_p)^2 + (y-y_p)^2 = r^2
Atau untuk y: 
y = y_p \pm \sqrt{r^2 - (x-x_p)^2}
Pendekatan langsung (Brute Force) menggunakan persamaan ini memerlukan operasi perkalian, akar kuadrat (SQRT), dan pembulatan (ROUND) yang mahal. Ini juga dapat menghasilkan *gap* piksel karena perubahan gradien.

Algoritma Midpoint Circle (Bresenham's Circle Algorithm)

Algoritma Midpoint lebih efisien karena menghindari operasi floating-point. Ia memanfaatkan simetri delapan titik lingkaran. Sebuah lingkaran dibagi menjadi delapan oktan (masing-masing $45^\circ$). Dengan menghitung piksel di satu oktan (misalnya dari $x=0$ hingga $x=y$ di oktan pertama), tujuh titik simetri lainnya dapat ditentukan secara otomatis.

Piksel yang digambar akan muncul di sini. 
Prinsip Midpoint Circle Algorithm (untuk pusat (0,0)):
1.  Pilih jari-jari r dan titik pusat (xp, yp).
2.  Titik awal: (0, r).
3.  Hitung parameter keputusan awal: p0 = 1 - r (jika r integer) atau p0 = 5/4 - r (jika r pecahan). 
4.  Jika pk < 0, titik berikutnya adalah (xk+1, yk). pk+1 = pk + 2xk+1 + 1. 
5.  Jika pk >= 0, titik berikutnya adalah (xk+1, yk-1). pk+1 = pk + 2xk+1 + 1 - 2yk+1. 
6.  Gunakan simetri delapan titik untuk memplot semua piksel. 
7.  Ulangi hingga x >= y.
8.  Geser piksel ke pusat (xp, yp): x = x + xp, y = y + yp.

2. Algoritma Pembentukan Elips

Elips adalah modifikasi dari lingkaran dengan penambahan sumbu minor (r_y) dan sumbu mayor (r_x). Persamaan elips dengan pusat (x_c, y_c) adalah: 

((x-x_c)/r_x)^2 + ((y-y_c)/r_y)^2 = 1
Algoritma Midpoint juga dapat digunakan untuk elips, dibagi menjadi dua bagian berdasarkan kemiringan (slope) dan memanfaatkan simetri empat kuadran.

Algoritma Midpoint Ellipse

Mirip dengan algoritma lingkaran, algoritma ini menggunakan parameter keputusan untuk memilih piksel berikutnya, memanfaatkan simetri untuk mengurangi perhitungan.

Piksel yang digambar akan muncul di sini. 
Prinsip Midpoint Ellipse Algorithm (untuk pusat (0,0)):
1. Tentukan sumbu mayor rx, sumbu minor ry, dan pusat (xc, yc).
2. Mulai dari titik (0, ry). 
3. Region 1 (slope < 1):
   Parameter keputusan awal: p1_0 = ry^2 - rx^2 * ry + 1/4 * rx^2. 
   Jika p1k < 0, titik berikutnya (xk+1, yk). p1_k+1 = p1_k + 2ry^2 * xk+1 + ry^2. 
   Jika p1k >= 0, titik berikutnya (xk+1, yk-1). p1_k+1 = p1_k + 2ry^2 * xk+1 - 2rx^2 * yk+1 + ry^2. 
   Iterasi sampai 2ry^2 * x >= 2rx^2 * y. 
4. Region 2 (slope > 1):
   Parameter keputusan awal dihitung dari titik akhir Region 1.
   Jika p2k > 0, titik berikutnya (xk, yk-1). p2_k+1 = p2_k + 2rx^2 * yk+1 + rx^2. 
   Jika p2k <= 0, titik berikutnya (xk+1, yk-1). p2_k+1 = p2_k + 2ry^2 * xk+1 - 2rx^2 * yk+1 + ry^2. 
   Iterasi sampai y = 0.
5. Gunakan simetri empat kuadran (x,y), (-x,y), (x,-y), (-x,-y) untuk memplot semua piksel.
6. Geser piksel ke pusat (xc, yc): x = x + xc, y = y + yc.
