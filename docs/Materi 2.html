Grafika Komputer: Titik dan Palet Warna

1. Piksel dan Resolusi

Piksel adalah satu titik data dalam gambar digital atau di monitor, dan pengukuran PPI menunjukkan jumlah piksel yang terkandung dalam gambar atau layar. Resolusi adalah kehalusan detail dalam gambar bitmap dan diukur dalam piksel per inci (ppi). Semakin banyak piksel per inci, semakin besar resolusinya.

Resolusi Yang Dimasukkan: 111x111 piksel.

Rumus Jumlah Piksel: Jumlah Piksel = Lebar*Tinggi

Jumlah Total Piksel: 12321 piksel.

Perangkat umumnya mengandalkan piksel persegi. Monitor modern terdiri dari ribuan piksel yang terbagi dalam baris dan kolom. Contoh resolusi maksimum monitor adalah 1024x768, yang berarti 786.432 piksel. Semakin tinggi jumlah piksel, semakin tajam gambar yang ditampilkan.

3. Konversi Warna RGB dan HSV

Warna pada citra digital direpresentasikan dengan tiga warna: merah, hijau, dan biru (model RGB). Besaran nilai ketiga warna bergantung pada bit yang digunakan. Semakin banyak bit, semakin banyak rentang nilai yang tersedia. Model RGB adalah model warna aditif di mana merah, hijau, dan biru dicampur untuk membentuk warna berbeda. Jika ketiga warna ditumpangkan dengan intensitas paling kecil menghasilkan hitam, dan intensitas cahaya penuh menghasilkan putih. Model RGB digunakan pada tampilan digital (TV, komputer, kamera digital).

Model HSV (Hue, Saturation, Value) menentukan warna berdasarkan Hue (corak warna), Saturation (kemurnian warna), dan Value (kecerahan). Model ini menyerupai persepsi warna manusia dan mudah untuk menyesuaikan warna dengan terang dan kecerahannya. Hue adalah sudut pada lingkaran warna (0° hingga 360°). Saturasi dinyatakan dalam persentase (0%-100%), di mana 100% berarti jenuh dan 0% berarti abu-abu netral. Value juga dalam persentase (0%-100%), di mana 0% berarti hitam dan 100% berarti spektrum murni.

Konversi RGB ke HSV

Rumus Konversi RGB ke HSV:

R' = R / 255
G' = G / 255
B' = B / 255

C_{max} = max(R', G', B')
C_{min} = min(R', G', B')
Delta = C_{max} - C_{min}

Hue (H):
Jika Delta = 0 maka H = 0° (Atau seringkali dianggap tidak terdefinisi, tapi secara teknis 0° digunakan sebagai default)
Jika C_{max} = R' maka H = 60° * (G' - B' / Delta (mod 6))
Jika C_{max} = G' maka H = 60° * ((B' - R' / Delta) + 2)
Jika C_{max} = B' maka H = 60° * ((R' - G' / Delta) + 4)

Saturation (S):
Jika C_{max} = 0 maka S = 0 (ini berarti warna hitam, tidak ada saturasi)
Jika C_{max} ≠ 0 maka S = Delta / C_{max}

Value (V):
V = C_{max}

Konversi HSV ke RGB

Rumus Konversi HSV ke RGB:

S {input} = S / 100
V {input} = V / 100

C = V * S
X = C * (1−∣(H/60°)(mod2)−1∣)
m = V - C

Jika 0°≤ H <60° : (R′ ,G′ , B′)=(C, X, 0)
Jika 60°≤ H <120° : (R' ,G' ,B') = (X, C, 0)
Jika 120°≤ H <180° : (R' ,G' ,B') = (0, C, X)
Jika 180°≤ H <240° : (R' ,G' ,B') = (0, X, C)
Jika 240°≤ H <300° : (R' ,G' ,B') = (X, 0, C)
Jika 300°≤ H <360° : (R' ,G' ,B') = (C, 0, X)

R = (R' + m) * 255
G = (G' + m) * 255
B = (B' + m) * 255
