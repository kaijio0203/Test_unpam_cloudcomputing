Aplikasi Grafika Komputer

1. Simulasi Aplikasi Animasi

Aplikasi grafika sering digunakan untuk membuat animasi. Simulasi ini terinspirasi dari contoh kode Java2D pada materi kuliah, di mana sebuah karakter dan teks dianimasikan. Di sini, Anda dapat mengontrol berbagai parameter animasi secara *real-time*.

2. Aplikasi Penggambaran Bentuk Statis

Selain animasi, aplikasi grafika juga sering digunakan untuk menggambar bentuk-bentuk statis. Simulasi ini akan menggambar sebuah poligon statis sesuai dengan contoh kode Java (`uas.java`) dari materi kuliah.
