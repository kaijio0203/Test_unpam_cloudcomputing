2. Transformasi Translasi

Translasi adalah pergeseran objek. Setiap titik pada objek digeser sejauh vektor translasi $(t_x, t_y)$.

$x' = x + t_x$,  $y' = y + t_y$

3. Transformasi Rotasi

Rotasi memutar objek mengelilingi sebuah titik pivot $(x_p, y_p)$ sebesar sudut $\theta$. Sudut positif berarti berlawanan arah jarum jam.

$x' = x_p + (x - x_p)\cos\theta - (y - y_p)\sin\theta$
$y' = y_p + (x - x_p)\sin\theta + (y - y_p)\cos\theta$
