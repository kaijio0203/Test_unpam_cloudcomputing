Grafika Komputer: API Grafis dan Pengolahan Citra Digital

1. API Grafis dan Primitif Geometris

OpenGL (Open Graphics Library) adalah sebuah Antarmuka Pemrograman Aplikasi (API) untuk grafika 2D dan 3D. API ini menjadi standar industri yang digunakan dalam berbagai aplikasi. Sebaliknya, pengguna harus membangun model yang diinginkan dari sekumpulan kecil primitif geometris, yaitu titik, garis, dan poligon.

2. Pengolahan Citra Digital

Citra digital adalah representasi gambar yang dapat diolah oleh komputer. Citra ini dibentuk melalui dua proses utama: sampling (menentukan resolusi spasial) dan kuantisasi (menentukan jumlah warna).
